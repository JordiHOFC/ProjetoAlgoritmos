Neste codigo é feita a implementação dos algoritmos de Ordenaçã, insertionSort, SelectionSort, MergeSort,ShellSort,QuickSort e HeapSort.
Objetivo deste trabalho é analisar o funcionamento de cada um desses algoritmos para varios tipos de instancias, de 100 a 1000000, de forma aleatoria, crescente e decrescente.
